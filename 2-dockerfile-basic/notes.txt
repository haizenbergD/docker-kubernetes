# Build docker image from Dockerfile
`docker build .`

# Tagging an image
`docker build -t {docker-id/repo:version} .`
and then `docker run {docker-id/repo:version}`

Examples:
build -  `docker build -t derkachd/redis:1.0 .`,
run - `derkachd/redis:1.0`

# Buildind an image from container

Staring apline - docker run -it alpine sh
Attach to the container and install gcc + redis.
Assigning default command to the running image - docker commit -c 'CMD ["redis-server"]' 272d63051b82
Using an image - docker run {container_hash}
